# Portfolio template (one-page визитка)

Шаблон одностраничного портфолио/визитки: статический HTML, тёмная палитра «салун», фото в круге, контакты. Подходит для деплоя на Vercel без сборки.

## Что внутри

- **index.html** — одна страница со встроенными стилями: герой (фото + имя + подзаголовок), блок «Обо мне», контакты (email, GitHub, Telegram).
- **img/** — положите сюда своё фото как `hero.jpg` (или измените путь в HTML).
- **docs/** — краткий workflow: handoff, возвращение к проекту.
- **SKILL.md** — навык для Cursor: как создавать такие портфолио по шагам (данные, git, деплой).

## Как использовать

1. Склонируйте репозиторий или скопируйте файлы в свой проект.
2. Замените в **index.html** плейсхолдеры: имя, подзаголовок, текст «Обо мне», email, ссылки на GitHub и Telegram.
3. Добавьте своё фото в **img/hero.jpg** (рекомендуется локальный файл, не внешняя ссылка с ограничением по referrer).
4. По желанию настройте палитру в `<style>` (цвета фона, текста, рамки фото).
5. Инициализируйте git, создайте репозиторий (приватный или публичный по вашему выбору), запушьте.
6. Задеплойте на Vercel: `vercel --prod` из корня проекта или импорт репо в Vercel Dashboard.

## Подключение к API (дополнение)

Если вы добавляете чат-бота или форму с отправкой на сервер, общий путь интеграции с внешними API описан в [docs/api-integration.md](docs/api-integration.md): секреты в переменных окружения, один endpoint на один провайдер, смена API без правки фронта. Подходит для воркшопов и обучения.

## Зависимости

Нет. Только браузер и (для деплоя) аккаунт Vercel и при необходимости GitHub.

## Лицензия

Шаблон можно свободно использовать и менять. Указание автора приветствуется, но не обязательно.
